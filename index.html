What is GitHub?

GitHub is the cloud-based platform for version control and collaboration. It lets you and others work together on projects from anywhere.

  • Host and review code
  • Manage projects
  • Collaborate with teams

What is DevOps?

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the development lifecycle and deliver high-quality software continuously.

  • Continuous Integration / Delivery (CI/CD)
  • Infrastructure as Code (IaC)
  • Monitoring and Logging
